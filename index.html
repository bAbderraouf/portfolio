2D Tetris Game (C++)

[2025]

Tetris game I developed using C++, Visual Studio, and the raylib library for graphics and input handling.
The project demonstrates effective use of C++ OOP features such as polymorphism, inheritance, interfaces, abstraction, agregation, abstract base classes, and memory management.

The game structure is devided into 4 principal parts contained in 4 functions.

  • Init() : setup of game variables, grid and window.
  • Input() : checking user inputs for each cycle of the game loop (for each frame).
  • Update() : for each cycle we have to update the game logic (mouvements, rotations, posistions..).
  • Draw() : after each update we have to draw objects on the screen.
